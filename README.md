# README
# AI Career Mentor & ATS Resume Analyzer

## Overview

AI Career Mentor & ATS Resume Analyzer is a web application that evaluates resumes, calculates ATS scores, identifies missing skills, recommends suitable career roles, generates learning roadmaps, and provides interview preparation questions.

## Features

* ATS Resume Score
* Skills Detection
* Missing Skills Analysis
* Career Recommendations
* Learning Roadmap Generation
* Resume Improvement Suggestions
* Interview Questions

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the project.
2. Open the project folder in VS Code.
3. Open index.html using Live Server.
4. Paste your resume.
5. Enter your career interest.
6. Click Analyze Resume.
